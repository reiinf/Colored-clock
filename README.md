Это расширение для FF, показывает иконку часов и в зависимости от выставленного интервала меняет свой цвет. 

Так же можно накапливать исторические данные по времени


Установка:
Firefox → about:debugging → This Firefox → Load Temporary Add-on → выбрать ff-extension.zip (или manifest.json внутри папки)
